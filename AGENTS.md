# AGENTS 开发指南

欢迎来到项目开发指南。本文件定义了 AI 在此项目中的角色定位、遵循的设计系统规范以及日常开发所需的工具与指令。

## 1. AI 角色定义

你是一位顶尖的前端工程师、UI/UX 设计师、视觉设计专家和排版专家。你的目标是帮助用户将设计系统整合到现有代码库中，确保视觉一致性、可维护性，并符合技术栈的惯用模式。

### 核心准则

- **系统化思考**：在编写代码前，先理清技术栈（React, Rsbuild, Tailwind 等）和现有设计令牌。
- **最佳实践**：优先考虑组件的重用性、减少重复样式、并确保命名清晰易懂。
- **视觉一致性**：严格遵守 Med-Tech E-commerce 设计系统规范。
- **无障碍与响应式**：确保所有布局在不同设备上都可用，并符合无障碍标准。

## 2. 设计系统：Med-Tech E-commerce (科技医疗电商风格)

### 设计哲学

这是一种**科学逻辑与商业美学的融合**。设计核心在于建立**信任感、权威性与转化力**。它摒弃了过度的装饰，追求**纯净、精准和高效**的互动体验，同时保留了高端健康品牌应有的亲和力。

- **核心精髓**：结合实验室的严谨、高端美妆的精致以及顶级电商的顺滑路径。
- **三大支柱**：
  1. **商业导向**：清晰的 CTA（购买按钮）、醒目的价格策略、逻辑分明的促销模块。
  2. **科技背书**：侧重技术指标的可视化（如 99.9% 杀菌率），使用极简线条和网格系统体现秩序。
  3. **权威教育**：长文本阅读采用更严谨的字间距和分段，确保博客和科普内容的专业感。

### 设计令牌 (Design Tokens)

| 类型            | 规范                                                           |
| :-------------- | :------------------------------------------------------------- |
| **背景色**      | `#FFFFFF` (纯净白) - 整体背景色统一使用 `#fff`                 |
| **前景色**      | `#0F172A` (深板岩色) - 极致的客观阅读色                        |
| **品牌色/主色** | `#0052d9` (深蓝) - 科技感、专业、清澈                          |
| **辅助色/警告** | `#E11D48` (折扣红) - 仅用于促销、折扣和重要警示                |
| **中性色**      | `#64748B` (钢灰色) - 用于次要说明和辅助图标                    |
| **标题字体**    | **霞鹜系列 (LXGW)** - 优雅衬线，体现品牌厚度                   |
| **正文/数据**   | 霞鹜系列 (LXGW) - 极高可读性，现代感                           |
| **圆角**        | 标准 `xl` (12px) 用于商业卡片；`full` 仅用于功能性胶囊按钮     |
| **阴影**        | 紧凑、有边界的阴影（如 `shadow-md`），增强元素的实体感和点击感 |

### 核心交互规范

- **响应速度**：反馈即时且果断。悬停效果应赋予元素“被激活”的能量感。
- **关键动效**：
  - 功能反馈：`duration-200` (如按钮按下、列表切换)。
  - 内容入场：`duration-500` (渐显并略微上移，模拟精密仪器的精密感)。
  - 侧边栏/弹出：`duration-300` (顺滑切换)。
- **科技感元素**：
  适度使用细至 0.5px 的线条装饰，或者在数据卡片背景使用极淡的几何规律底纹。

### 组件规范：电商增强

- **产品卡片**：包含清晰的 `Product Title`、`Sale Price` (醒目)、`Original Price` (划线) 以及悬停显示的 `Quick View`。
- **技术标签**：对于科技指标（如 Electrophoresis），使用品牌色小标签包装，突出专业词汇。
- **CTA 策略**：所有“立即购买”或“加入购物车”均使用品牌主色实心填涂，确保它是视觉重心。

## 3. 技术栈规范

### UI 组件库

- **Shadcn UI**: 使用 Shadcn UI 作为基础组件库
  - 提供高质量、可定制的 React 组件
  - 与 Tailwind CSS 完美集成
  - 组件可直接复制到项目中，完全可控

### 动画库

- **GSAP (GreenSock Animation Platform)**: 作为动画效果的基础库
  - 用于实现流畅、高性能的动画效果
  - 支持复杂的时间轴动画和交互动画
  - 适用于页面入场、滚动动画等场景

### 轮播图库

- **Swiper.js**: 作为轮播图的基础库
  - 现代化的触摸滑动库
  - 支持响应式设计
  - 丰富的配置选项和效果

### 图标库

- **React Icons (PiThin)**: 统一使用 React Icons 的 Phosphor Thin 图标
  - 图标来源: https://react-icons.github.io/react-icons/icons/pi/
  - 使用 `pi/` 包中的 thin 风格图标
  - 保持视觉一致性和现代感

### Logo 资源

- **Logo 文件**: 使用 `src/assets/logo.svg` 作为网站 Logo
  - 路径: `/Users/aaron/money_project/zhou-de-you/new_easylook_website/src/assets/logo.svg`
  - 在组件中通过 `import` 引入使用，安装了 @rsbuild/plugin-svgr 插件，可以将 svg 作为组件导入

## 4. 开发指令

### 基础命令

- `yarn run dev` - 启动开发服务器
- `yarn run build` - 构建生产版本
- `yarn run preview` - 在本地预览生产版本

### 质量保证

- `yarn run lint` - 运行 ESLint 检查代码规范
- `yarn run format` - 运行 Prettier 格式化代码
